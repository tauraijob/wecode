# Logo Upload Instructions

## How to Upload Your Logo

### Option 1: Direct File Upload (Recommended)
1. Place your logo file directly in the `public/logo/` directory
2. Name it `logo.png` or `logo.svg` (preferred formats)
3. Supported formats: PNG, SVG, JPG, WEBP
4. Recommended size: 200x50px or similar aspect ratio
5. The logo will automatically appear in the website header

### Option 2: Via Admin Panel
1. Log in as admin
2. Go to Admin Dashboard â†’ Settings (if available)
3. Use the logo upload feature
4. The logo will be saved to `public/logo/logo.png`

## File Location
- **Directory:** `public/logo/`
- **Filename:** `logo.png` or `logo.svg`
- **URL:** `/logo/logo.png` or `/logo/logo.svg`

## Logo Specifications
- **Format:** PNG (with transparency) or SVG (recommended)
- **Size:** 200x50px or similar (height should be around 40-60px)
- **Background:** Transparent (recommended)
- **Colors:** Should work on dark backgrounds (navy-950)

## Current Logo
The website currently uses a text-based logo "WZ" in a box. Once you upload a logo file, it will replace the text logo automatically.
